# GM Path – 1400 ao GM (PWA)

App de treino de xadrez para ir de 1400 até nível de Grande Mestre.

## Como publicar no GitHub Pages (3 minutos)

1. Crie um repositório novo no GitHub chamado `gmpath` (público)
2. Faça upload destes arquivos: `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
3. Vá em Settings > Pages > Build and deployment > Source: Deploy from a branch > Branch: main > / (root) > Save
4. Aguarde 1 min. Seu link será: `https://SEU-USUARIO.github.io/gmpath/`

## Instalar no celular
- Android (Chrome): abra o link > menu ⋮ > Instalar aplicativo
- iPhone (Safari): Compartilhar > Adicionar à Tela de Início

Funciona offline após primeira abertura.
